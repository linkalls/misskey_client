## misskey client
A client for Misskey API　written in Dart.


## 使い方
.sample.envを.envにリネームして、MisskeyのインスタンスのURLとアクセストークンを入力してください。
そのあと、以下のコマンドを実行してください。
```shell
flutter pub get
```
そのあとに.envを読み込むために以下のコマンドを実行してください。
```shell
dart run enven
```
そのあと、以下のコマンドを実行してビルドしてください
```shell
flutter build apk
```
